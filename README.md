# 介绍
这是一个基于vue全家桶仿Mac桌面的小项目
* vue + iconfont图标完成基本界面
* 利用vue-layer实现弹框层
* 利用moment实现时间显示
* 基于localStorage实现壁纸更改

# 线上预览地址
https://hdong-math.github.io/

# 运行
下载依赖

//npm i

启动开发环境

//npm run serve

构建

//npm run build
